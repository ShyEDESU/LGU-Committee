## ğŸ¯ Common Tasks

### View Committee Information
1. Click "Committee Profiles & Membership" in sidebar
2. View committees in grid on "Committees" tab
3. Click "View Details" to see full information

### Schedule a Meeting
1. Go to "Committee Meetings Management"
2. Click "Schedule Meeting" button
3. Fill in meeting details
4. Confirm and save

### Create an Action Item
1. Navigate to "Action Items & Follow-Ups"
2. Click "New Action Item" button
3. Fill in item details, assign person, set deadline
4. Save and track progress

### Generate a Report
1. Go to "Committee Reports & Recommendations"
2. Click "New Report" button
3. Select report template
4. Add content and recommendations
5. Publish or save as draft

### Track a Referral
1. Open "Referral Tracking & Handling"
2. View all active referrals in table
3. Click referral to see details
4. Update status or deadline as needed

### Check Notifications
1. Click bell icon (ğŸ””) in header
2. View dropdown list of notifications
3. Click notification to view details
4. Use "View All Notifications" for full list
